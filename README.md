# 🌐 EvntSphere

**EvntSphere** is a modern, elegant landing page designed to help users discover artists and events globally. Powered by clean HTML & CSS, it's a responsive and stylish front-end prototype.

## 🎯 Features

- 🎨 Elegant UI with animation and hover effects
- 🌍 Global discovery theme
- 🔍 Integrated search card for artist lookup
- 🧑‍🎤 Verified artist highlights
- 📱 Responsive layout for all screen sizes


## 🛠️ Tech Stack

- HTML5
- CSS3 (with Google Fonts & Animations)
